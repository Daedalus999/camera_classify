# ClimbSnap — 攀岩照片/视频分类管理 App

## 产品需求文档 (PRD)

### 1. 产品概述

ClimbSnap 是一款面向攀岩爱好者的照片与视频分类管理应用。用户可以按日期和攀岩馆（地点）对拍摄的照片和视频进行分类整理，并在应用内流畅浏览，支持视频倍速播放。所有媒体文件始终存储在手机本地，应用仅维护分类索引，不进行任何云端上传。

### 2. 目标用户

- 攀岩爱好者，需要按训练日期和场馆整理攀岩照片/视频
- 运动记录者，希望通过日历快速回顾某天某地的训练内容
- 对隐私敏感的用户，不希望媒体文件上传至云端

### 3. 核心功能

#### 3.1 日历视图

| 项目 | 说明 |
|------|------|
| 月历展示 | 以月为单位展示日历，有媒体记录的日期显示标记点 |
| 日期选择 | 点击日期查看该日所有上传的照片和视频 |
| 快速导航 | 支持左右滑动切换月份，点击"今天"快速回到当前日期 |

#### 3.2 媒体上传与分类

| 项目 | 说明 |
|------|------|
| 选择媒体 | 从手机相册选择照片或视频 |
| 指定日期 | 上传时选择关联日期（默认当天） |
| 指定地点 | 上传时选择或新建地点（如"岩时攀岩馆"） |
| 本地存储 | 媒体文件不复制不移动，仅记录文件路径引用 |
| 批量上传 | 支持一次选择多张照片/多个视频 |

#### 3.3 媒体浏览

| 项目 | 说明 |
|------|------|
| 网格浏览 | 照片和视频以缩略图网格展示 |
| 全屏查看 | 点击缩略图进入全屏查看，照片支持缩放 |
| 视频播放 | 内置视频播放器，支持播放/暂停/进度拖拽 |
| 倍速播放 | 视频支持 0.5x / 1x / 1.5x / 2x / 3x 倍速 |
| 左右滑动 | 全屏模式下左右滑动切换上一张/下一张 |

#### 3.4 地点管理

| 项目 | 说明 |
|------|------|
| 自定义地点 | 用户可自由创建地点名称（如"岩时攀岩馆""首岩攀岩"） |
| 地点列表 | 查看所有已创建的地点 |
| 按地点筛选 | 在日历日期下按地点筛选媒体 |
| 编辑/删除 | 支持编辑地点名称和删除地点 |

### 4. 非功能需求

| 项目 | 说明 |
|------|------|
| 隐私 | 所有数据存储在本地，无网络请求，无云同步 |
| 性能 | 大量媒体时列表流畅滚动，使用虚拟化列表 |
| 平台 | 支持 iOS 和 Android |
| 离线 | 完全离线可用 |

### 5. 页面结构

```
Tab 导航
├── 📅 日历 (首页)
│   ├── 月历视图
│   └── 日期媒体列表（按地点分组）
├── 📍 地点
│   ├── 地点列表
│   └── 地点详情（该地点所有媒体）
└── ⚙️ 设置
    └── 关于 / 数据管理
```

### 6. 交互流程

#### 上传流程
```
日历页 → 选择日期 → 点击"+"按钮 → 选择地点（或新建）→ 从相册选择照片/视频 → 完成
```

#### 浏览流程
```
日历页 → 选择日期 → 查看该日媒体网格 → 点击进入全屏 → 左右滑动浏览 → 视频可调倍速
```
